# Kc-Portfolio-Website

This is the official portfolio website for Kc Casipit, showcasing his skills as a web developer. It is a fully responsive, multi-section website built with HTML, CSS, and custom JavaScript, designed to showcase his projects, services, and professional background.

## ðŸš€ How to Run Locally

To explore the website on your local machine, please follow these steps:

1.  Open your terminal (Git Bash or CMD).
2.  Clone the repository into a new folder named `Kc-Casipit-Personal-Portfolio`:
    ```bash
    git clone [https://your-repository-url-here.git](https://your-repository-url-here.git) Kc-Casipit-Personal-Portfolio
    ```
3.  Open the `Kc-Casipit-Personal-Portfolio` folder in Visual Studio Code.
4.  Open the `index.html` file.
5.  Right-click on the file and select **"Open with Live Server"**.

> **Important Note:** You **must** use the **Live Server** extension by Ritwick Dey in VS Code. *This is critical to ensure that all JavaScript elements, especially the parallax effects and navigation, work properly and are displayed as intended.*
