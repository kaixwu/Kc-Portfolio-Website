Kc Best Sales

A responsive e-commerce site for a local auto parts seller.

View Live Site

Project Details

The Challenge

The client needed a modern, responsive website to showcase their new product line and drive online sales. The old site was slow and not mobile-friendly.


My Solution

I built a custom, mobile-first website using HTML, CSS, and JavaScript. I optimized all images for fast loading, which improved their site speed by 40%.

Tech Stack

Wordpress

Used WordPress with Elementor to build a custom theme, allowing the client to easily update products and content without any code.

JavaScript

Implemented custom JavaScript for the interactive FAQ accordion, product image gallery, and a smooth-scrolling navigation bar.

Adobe Photoshop

All product photos and hero banners were edited, color-corrected, and compressed in Photoshop for a professional look and fast load times.
